Body Mass Index
===============

Imperial BMI Formula
--------------------
In order to determinate BMI:

* Weight :kilogrmas.
* Height :meters.

### Public Methods

- weight();
- height();

Example:

bmi.weight(80,'k');
bmi.height(1.7,'m');

- calc();

Example:

bmi.calc();
